# MCP Test 專案

## 專案簡介

這是一個簡單的 Node.js 驗證系統測試專案，主要用於展示基本的模組化設計和使用者驗證功能。專案包含了驗證邏輯模組和主要測試程式，展示了如何將功能拆分成獨立模組並進行引用。

## 檔案結構樹狀圖

```
mcp-test/
│
├── auth.js          # 驗證邏輯模組
├── index.js         # 主程式進入點
└── README.md        # 專案說明文件
```

## 檔案功能說明

### auth.js
**功能：** 驗證邏輯模組

- 提供使用者登入驗證功能
- 實作 `login(username, password)` 函數
- 驗證使用者名稱和密碼是否正確
- 回傳包含驗證結果和訊息的物件
- 使用 `module.exports` 匯出函數供其他模組使用

**匯出功能：**
- `login(username, password)` - 執行登入驗證並回傳結果

### index.js
**功能：** 主程式進入點

- 引用 `auth.js` 模組中的登入功能
- 測試登入功能的執行
- 使用 `require('./auth')` 引入驗證模組
- 呼叫 `login()` 函數並輸出測試結果

**依賴關係：**
- 引用：`auth.js` (透過 `require('./auth')`)

## 模組關聯圖

```
index.js  ──→  auth.js
(主程式)        (驗證模組)
   │              │
   │              ├─ login()
   │              └─ 驗證邏輯
   │
   └─ 測試登入功能
```

## 執行方式

```bash
node index.js
```

## 功能流程

1. `index.js` 引入 `auth.js` 模組
2. 呼叫 `login('admin', '1234')` 進行驗證
3. 接收驗證結果並輸出訊息到控制台
